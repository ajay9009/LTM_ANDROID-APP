JavaScript Type Conversion

Number() converts to a Number, String() converts to a String, Boolean() converts to a Boolean.

JavaScript Data Types

In JavaScript there are 5 different data types that can contain values:
  • string
  • number
  • boolean
  • object
  • function

There are 6 types id objects:

  • Object
  • Date
  • Array
  • String
  • Number
  • Boolean
And 2 data types that cannot contain values:
  • null
  • undefined

Converting Strings to Numbers

The global method Number() can convert strings to numbers. Strings containing numbers (like "3.14") convert to numbers (like 3.14). Empty strings convert to 0. Anything else converts to NaN (Not a Number).
